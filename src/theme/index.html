Gulp Sass Starter Setup

Looks like you've loaded everything up. Now try:

  1. In your command line editor (Terminal on Mac), navigate to this project folder and run the command 'gulp', followed by 'gulp default'.
  2. Next, open your code editor and find '/src/sass/frontend.scss' and change the section .main-success.
  3. Come back to this page - you shouldn't have to reload to see the new colour.
  4. If you don't see any changes, retry the above steps and make sure you've followed the tutorial carefully.

✔ Nice work, you're all set!

Gulp is set to watch your Sass, JS and HTML files as you develop. You can now remove any stylesheet files you don't need and work on building out your website/app as you please.
